# Wayfinder Roadmap

## Current Phase

Skeleton Certified candidate.

## Next Milestones

1. Validate constitutional skeleton.
2. Prepare Interface Certified placeholders for semantic artifacts,
   governance artifacts, extension admission, policy language, and model
   compatibility.
3. Plan alpha work only after validation gates and approval.
4. Resolve RFC-0001 before any Wayfinder compiler transfer begins.
5. Participate in `E2.PH3.EPIC-12` under the Foundation Transfer Protocol only
   after destination, compatibility, equivalence, rollback, and evidence gates
   pass.

## Constraints

Wayfinder must remain semantic authority only. It must not become a runtime
service, persistence service, UI, deployment system, contract packager, SDK
generator, telemetry collector, or assistant implementation.

The Wayfinder to Scribe compiler extraction moves compiler implementation
responsibility only. Wayfinder retains canonical semantics, WPL semantics,
English Index, terminology, and semantic relationships. Ecosystem order,
validation, evidence, exit criteria, and success metrics are linked from the
Polaris `roadmap/implementation-program.md` entry `E2.PH3.EPIC-12` and are not
duplicated here.
