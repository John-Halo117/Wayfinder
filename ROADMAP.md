# Wayfinder Roadmap

## Current Phase

Skeleton Certified candidate.

## Next Milestones

1. Validate constitutional skeleton.
2. Prepare Interface Certified placeholders for semantic artifacts,
   governance artifacts, extension admission, policy language, and model
   compatibility.
3. Plan alpha work only after validation gates and approval.

## Constraints

Wayfinder must remain semantic authority only. It must not become a runtime
service, persistence service, UI, deployment system, contract packager, SDK
generator, telemetry collector, or assistant implementation.
