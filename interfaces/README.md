# Public Interface Placeholders

Wayfinder public interface placeholders are conceptual and transport-neutral.

Approved public surfaces:

- canonical specifications;
- registries;
- semantic queries as artifacts;
- RFC and ADR review boundaries;
- Commons-published mechanical forms of Wayfinder-owned semantics.

ADR-0009 constrains generic command, query, event, plugin, resource,
configuration, dependency, and observability language. Non-applicable generic
surfaces are reserved and inactive.
