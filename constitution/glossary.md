# Glossary

## Capability

A stable architectural verb. Implementations may change while the capability
remains.

## Contract

Shared language with explicit interfaces and schemas. Contracts contain no
implementation.

## Domain

A real-world composition of engines.

## Engine

A composition of services implementing one unique architectural responsibility.

## Ephemeral

Disposable execution state used for working memory, simulation, recognition,
projection, or proof.

## Reality

Append-only canonical observations.

## Observation Source

A source-specific producer of canonical observation-shaped records.
Observation Sources discover, classify, parse, validate, and emit observations;
ARK preserves them.

## Source Relationship

An explicit relationship present in source data. ARK preserves Source
Relationships as evidence; WEAVE owns durable relationship topology.

## Import Profile

A bounded import posture defining limits, validation expectations, runtime
expectations, and replay behavior for one import class.

## Service

Reusable infrastructure shared by engines, domains, applications, or tooling.
