# Canonical Language

Canonical Language is Wayfinder's deterministic language normalization and
compression substrate.

It is derived from ARK-preserved reality. It is rebuildable. It is not reality,
knowledge, interpretation, reasoning, navigation, or summarization.

## Constitutional Boundary

Canonical Language receives ARK-preserved observations and produces derived,
traceable language artifacts for compiler, retrieval, and future AI consumers.

It must preserve:

- source traceability
- deterministic rebuildability
- source-agnostic normalization
- raw artifact immutability
- separation from durable knowledge

It must not:

- modify observations
- discard raw source content
- infer truth
- summarize meaning
- generate embeddings as canonical output
- become a Knowledge Graph
