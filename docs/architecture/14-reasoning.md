# 14 Reasoning

Reasoning evaluates evidence, interpretations, missions, constraints, policies,
and bearings to infer conclusions under uncertainty.

## Owns

- Argument structure.
- Contradiction handling.
- Confidence posture.
- Reasoned conclusions.

## Does Not Own

- Observation storage.
- View presentation.
- Commitments.
- Action execution.

## Current State

Reasoning is constitutionally defined but not implemented. Knowledge compiler
and generated reports detect candidate contradictions, but they are not a
Reasoning engine.

## AI Rule

AI is last, not first. Deterministic reasoning and provenance-preserving
evaluation should precede probabilistic/LLM reasoning.
