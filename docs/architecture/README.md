# Architecture Reference

This directory is the canonical architecture reference for Wayfinder. It
summarizes the system in dependency order and points to canonical owners in the
Constitution, Canon, Contracts, Services, Engines, and governance documents.

When an ADR references architecture, it should cite one or more numbered files
in this directory.
