# MICE

## Purpose

Owns commitment, consensus, coordination, and accountability.

## Owns

Commitments, agreements, consensus, delegation, accountability, negotiation, and human-agent coordination.

## Does Not Own

Planning, navigation, reality preservation, reasoning, or task execution.

## Inputs

Recommendations, identities, policies, parties, objectives, and reasoned conclusions.

## Outputs

Commitments, agreements, delegation records, accountability records, and consensus outcomes.

## Dependencies

Identity, Policy, Reasoning, Jarvis outputs, Permission Contracts.

## Consumers

Domains, internal applications, operations, Foundry, and operators.
