# NOMAD

## Purpose

Owns discovery.

## Owns

Discovered options, providers, resources, opportunities, and discovery evidence.

## Does Not Own

Navigation decisions, route selection, external system ownership, or capability language.

## Inputs

Capabilities, context, constraints, external references, and objectives.

## Outputs

Discovery candidates, provider options, and opportunity inventories.

## Dependencies

Capability Contracts, External Integrations, Identity, Policy.

## Consumers

Jarvis, ZWLib, Foundry, domains, and internal applications.
