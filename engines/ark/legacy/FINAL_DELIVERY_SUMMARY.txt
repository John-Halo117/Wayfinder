QUICK START

1. CONFIGURE
   cp .env.prod .env
   nano .env  # Set DOMAIN, passwords, paths

2. INITIALIZE (auto-generates all configs)
   bash init-prod.sh

3. VALIDATE
   bash validate-prod.sh

4. DEPLOY STAGING (test first)

6. DEPLOY PRODUCTION
   bash deploy-prod.sh production

7. MONITOR
   bash monitor-prod.sh monitor
