# VALOR

## Purpose

Owns value and risk evaluation.

## Owns

Value assessments, risk posture, tradeoff evaluation, and objective-relative significance.

## Does Not Own

Policy enforcement, navigation, commitments, measurement, or reality preservation.

## Inputs

Evidence, measurements, objectives, constraints, reasoning outputs, and policies.

## Outputs

Value/risk assessments, tradeoff surfaces, and evaluation findings.

## Dependencies

Reasoning, MIDAS, Evidence Contracts, Policy, CivPhys.

## Consumers

Jarvis, MICE, Blackwall, domains, and operators.
