# Review Views

Review views are deterministic projections over the candidate repository.

Implemented views:

- New Concepts
- Pending ADRs
- Glossary Candidates
- Constitutional Amendments
- Contradictions
- Duplicate Concepts
- Novel Ideas
- Recurring Principles
- Open TODOs
- Low Confidence Candidates
- High Impact Candidates

Views never change state. They are reproducible for the same repository
contents and governance timestamp.
