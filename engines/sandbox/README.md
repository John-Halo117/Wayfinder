# Sandbox

## Purpose

Owns simulation and bounded experimentation.

## Owns

Simulated worlds, counterfactuals, ephemeral trials, and safe exploratory scenarios.

## Does Not Own

Durable reality, commitments, production action, or promoted knowledge.

## Inputs

Hypotheses, constraints, context, transformation paths, and reasoned uncertainty.

## Outputs

Simulation results, counterfactual evidence, and proof candidates.

## Dependencies

ARK, Reasoning, ZWLib, Policy, Evidence Contracts.

## Consumers

Reasoning, Jarvis, Foundry, VALOR, and domains.
