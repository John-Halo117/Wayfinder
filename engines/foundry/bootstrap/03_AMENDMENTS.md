# 03 Amendments

## Responsibility

Collect constitutional changes and architectural decisions that modify or
clarify inherited doctrine.

## Inputs

- `docs/adrs/`
- `docs/first-contact/constitutional-change-log.md`
- Governance records that cite constitutional changes
- P0 Open Questions

## Outputs

- Amendment inventory
- Decision provenance map
- Amendment confidence entries

## Boundary

If no separate amendment ledger exists, record ADRs as observed decision
artifacts without declaring them equivalent to amendments.
