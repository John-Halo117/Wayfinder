# Foundry Bootstrap

Foundry begins by bootstrapping its constitutional worldview.

Bootstrap is mandatory. No engineering phase executes before bootstrap produces
the required session artifacts and records foundation completion.

## Order

```text
01_EISENGARTEN
-> 02_CONSTITUTION
-> 03_AMENDMENTS
-> 04_VOCABULARY
-> 05_ARCHITECTURE
-> 06_ROADMAP
-> 07_ENGINEERING
-> 08_PROMPT_STANDARD
-> 09_FOUNDATION_COMPLETE
```

## Required Artifacts

- Constitutional Model
- Concept Inventory
- Vocabulary
- Layer Model
- Dependency Model
- Traceability Model
- Engineering Rules
- Prompt Rules
- Confidence Report

## Law

Never ask a phase to rediscover information already represented by an inherited
artifact. Discovery occurs once. Reuse thereafter.
