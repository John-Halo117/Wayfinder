# 04 Vocabulary

## Responsibility

Establish canonical names, aliases, deprecated terms, and concept ownership.

## Inputs

- `canon/`
- `constitution/glossary.md`
- P0 Concept Inventory

## Outputs

- Vocabulary artifact
- Alias map
- Deprecated-name map
- Ambiguous ownership entries

## Boundary

Do not create a second definition for a concept that already has a canonical
home.
