# Foundry P2.5 Knowledge Integrity

P2.5 validates the Canonical Knowledge Graph and classifies uncertainty before
Architectural Intelligence begins.

This phase does not generate new knowledge. It verifies, reconciles, audits,
and records confidence using inherited bootstrap, P1, P2, repository, and
Knowledge graph artifacts.

## Outputs

- [Knowledge Integrity Report](knowledge-integrity-report.md)
- [Updated Open Questions](updated-open-questions.md)
- [Resolution Report](resolution-report.md)
- [Human Decision Queue](human-decision-queue.md)
- [Coverage Report](coverage-report.md)
- [Relationship Integrity Report](relationship-integrity-report.md)
- [Promotion Audit](promotion-audit.md)
- [Confidence Audit](confidence-audit.md)
- [Canonical Health Report](canonical-health-report.md)
- [P3 Readiness Report](p3-readiness-report.md)

## Boundary

Historical artifacts and generated Knowledge graph files remain unchanged.
Unresolvable uncertainty is preserved and prioritized rather than forced.
