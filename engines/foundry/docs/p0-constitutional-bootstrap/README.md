# Foundry P0 Constitutional Bootstrap

P0 constructs Foundry's constitutional worldview before repository discovery,
engineering extraction, implementation, or optimization.

## Source Set

- `WAYFINDER.md`
- `ROADMAP.md`
- `constitution/`
- `canon/`
- `docs/adrs/`
- `docs/architecture/`
- `docs/canonical-language/`
- `docs/governance/`
- `docs/programs/`

The standalone Eisengarten source was not found as a separate repository file
during this pass. The model therefore treats Eisengarten through the
constitutional foundation recorded in `WAYFINDER.md` and marks the missing
source as an open question.

## P0 Boundary

This artifact set is discovery-only. It does not inspect implementation code,
estimate implementation size, propose extraction opportunities, or alter
runtime behavior.
