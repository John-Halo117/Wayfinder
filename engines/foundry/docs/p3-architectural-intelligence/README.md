# Foundry P3 Architectural Intelligence

P3 analyzes inherited constitutional, repository, and knowledge artifacts to
discover architectural improvement opportunities.

This phase produces observations. It does not implement, prioritize, schedule,
plan migrations, or create pull requests.

## Outputs

- [Architecture Notebook](architecture-notebook.md)
- [Opportunity Catalog](opportunity-catalog.md)
- [Shared Capability Catalog](shared-capability-catalog.md)
- [Recursive Factorization Report](recursive-factorization-report.md)
- [Constitutional Alignment Report](constitutional-alignment-report.md)
- [Layer Violation Report](layer-violation-report.md)
- [Reuse Analysis](reuse-analysis.md)
- [Bundle Candidates](bundle-candidates.md)
- [Architectural Archaeology Report](architectural-archaeology-report.md)
- [P4 Readiness Report](p4-readiness-report.md)

## Boundary

P3 opportunities are not implementation tasks. P4 may group opportunities into
coherent bundles after inheriting this observation set.
