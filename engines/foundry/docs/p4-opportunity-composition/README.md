# Foundry P4 Opportunity Composition

P4 composes P3 architectural opportunities into coherent implementation
bundles.

Bundles are grouping artifacts. They are not priorities, schedules, migration
plans, or implementation tasks.

## Outputs

- [Bundle Catalog](bundle-catalog.md)
- [Bundle Dependency Graph](bundle-dependency-graph.md)
- [Shared Infrastructure Bundles](shared-infrastructure-bundles.md)
- [Capability Bundles](capability-bundles.md)
- [Migration Bundles](migration-bundles.md)
- [Bundle Notebook](bundle-notebook.md)
- [Bundle Validation Report](bundle-validation-report.md)
- [P5 Readiness Report](p5-readiness-report.md)

## Boundary

P4 discovers coherent bundles. P5 may prioritize them later.
