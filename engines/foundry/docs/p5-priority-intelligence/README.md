# Foundry P5 Priority Intelligence

P5 prioritizes P4 implementation bundles into a dependency-aware execution
order.

Priority artifacts rank and sequence bundles. They are not implementation
plans, schedules, migrations, or pull requests.

## Outputs

- [Priority Catalog](priority-catalog.md)
- [Recommended Execution Order](recommended-execution-order.md)
- [Dependency-Aware Roadmap](dependency-aware-roadmap.md)
- [TRISCA Evaluation Report](trisca-evaluation-report.md)
- [TriAvg Report](triavg-report.md)
- [Cost/Risk Analysis](cost-risk-analysis.md)
- [Priority Notebook](priority-notebook.md)
- [P6 Readiness Report](p6-readiness-report.md)

## Boundary

P5 decides priority. P6 may translate priority into migration planning later.
