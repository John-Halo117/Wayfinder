# Foundry P1 Repository Discovery

P1 maps the repository as it exists onto the P0 constitutional model.

## Inherited P0 Artifacts

- Constitutional Model
- Concept Inventory
- Layer Model
- Dependency Model
- Traceability Model
- Invariant Report
- Confidence Report
- Open Questions

## Source Set

- Repository directory structure
- Existing documentation
- Source code
- Build and deployment configuration
- Tests
- Generated documentation and generated knowledge artifacts

## Outputs

- [Repository Model](repository-model.md)
- [Repository Inventory](repository-inventory.md)
- [Repository Layer Map](repository-layer-map.md)
- [Ownership Graph](ownership-graph.md)
- [Dependency Graph](dependency-graph.md)
- [Interface Catalog](interface-catalog.md)
- [Coverage Report](coverage-report.md)
- [Unknown Components Report](unknown-components-report.md)
- [Confidence Report](confidence-report.md)

## P1 Boundary

This is a discovery artifact set. It records what exists, where it appears to
belong, and what remains unknown. It does not evaluate quality, recommend
changes, perform reuse analysis, move files, or implement anything.
