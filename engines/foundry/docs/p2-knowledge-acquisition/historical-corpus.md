# Historical Corpus

## Corpus Role

The historical corpus preserves evolution, abandoned names, legacy systems,
source conversations, and generated knowledge evidence.

## Source Groups

| Source group | Path | Notes |
| --- | --- | --- |
| Generated Knowledge | `Knowledge/` | Derived graph, reports, indexes, timelines. |
| First Contact | `docs/first-contact/` | ChatGPT export validation and evidence assimilation. |
| Legacy ARK | `engines/ark/legacy/` | Historical ARK runtime, operations, integrations, and Forge-origin material. |
| Foundry Legacy | `engines/foundry/legacy/` | Preserved Forge surfaces under Foundry ownership. |
| Validation Artifacts | `.wayfinder-validation/` | Local private validation outputs; not inspected deeply in P2. |

## Immutability Rule

Historical artifacts remain unchanged. P2 links, classifies, and reports on
them; it does not rewrite or delete them.
