# Promotion Report

## Promotion States

| State | Meaning |
| --- | --- |
| Observed | Extracted from source evidence. |
| Derived | Produced by deterministic normalization or graph compilation. |
| Candidate | Proposed for canonical consideration. |
| Promoted | Accepted by explicit governance or canonical source. |
| Superseded | Retained historically but replaced by a newer canonical record. |
| Rejected | Retained as evidence but not accepted as canonical. |
| Unresolved | Requires review or additional evidence. |

## Current Promotion Boundary

documents.

## Explainability Requirement

Every promotion decision must record:

- Source evidence.
- Provenance.
- Confidence.
- Reason.
- Previous state.
- New state.
