# Wayfinder Corpus

## Corpus Role

The Wayfinder corpus contains canonical project doctrine, architecture,
contracts, governance, roadmap, and Foundry bootstrap artifacts.

## Source Groups

| Source group | Path | Object types |
| --- | --- | --- |
| Constitution | `WAYFINDER.md`, `constitution/` | Concepts, principles, layers, invariants. |
| Canon | `canon/` | Vocabulary, aliases, ontology. |
| Architecture | `docs/architecture/` | Layers, responsibilities, dependencies. |
| ADRs | `docs/adrs/` | Events, decisions, constraints, relationships. |
| Programs | `ROADMAP.md`, `docs/programs/` | Projects, phases, requirements. |
| Contracts | `contracts/` | Protocols, capabilities, schemas. |
| Foundry Bootstrap | `engines/foundry/bootstrap/` | Boot sequence, boot policy, prompt rules. |
| Foundry P0/P1/P2 | `engines/foundry/docs/p*-*/` | Session artifacts and reports. |

## Promotion State

Canonical Wayfinder documents remain the authority for doctrine. The knowledge
graph may index, relate, and retrieve them but does not replace them.
