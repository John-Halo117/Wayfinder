# View Contracts

Every view result includes:

- view identifier
- view type
- generation timestamp
- view definition
- items
- groups
- validation report

Every view definition declares:

- purpose
- inputs
- outputs
- dependencies
- filters
- grouping
- sort order
- refresh strategy
- visibility
- permissions

Every item preserves:

- promotion record reference
- promoted artifact reference
- supporting observations
- source conversations
- source messages
- source Oracles
- compiler version
- promotion version
- confidence
- uncertainty
- review history
- promotion history
