# Knowledge Views

Knowledge Views are disposable projections over promoted knowledge and retrieval
results.

Views do not own knowledge, generate knowledge, promote knowledge, reason,
navigate, or execute actions. They arrange existing durable knowledge for an
audience, workflow, question, or review surface.

## Implemented Views

- Timeline View
- Architecture View
- Conversation View
- Concept View
- Decision View
- Glossary View
- Capsule View
- Objective View
- Review Queue View
- Search Results View
- Composed View
