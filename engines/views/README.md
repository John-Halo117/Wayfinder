# Views

## Purpose

Presents derived representations without owning source truth.

## Owns

Views, projections, read models, presentation-ready representations, and derived inspection surfaces.

## Does Not Own

Reality preservation, reasoning, interpretation, or storage backend behavior.

## Inputs

Durable knowledge, relationship topology, reasoned conclusions, projections, and context.

## Outputs

Consumable representations, projections, and read models.

## Dependencies

ARK, WEAVE, Reasoning, View Contracts, Storage.

## Consumers

Jarvis, Foundry, internal applications, domains, and operators.
