# NetWatch

## Purpose

Owns network-specific observation and response behavior.

## Owns

Network state interpretation, network findings, network response recommendations, and network watch posture.

## Does Not Own

Generic telemetry, external integration ownership, operations, or protection policy.

## Inputs

Network observations, health signals, external network references, and policies.

## Outputs

Network findings, network posture, and network response recommendations.

## Dependencies

ARK, Health Contracts, External Integrations, Policy.

## Consumers

Blackwall, Operations, Jarvis, Views, and domains.
