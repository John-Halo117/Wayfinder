# ZWLib

## Purpose

Owns transformation and affordance reasoning.

## Owns

Transformation paths, affordances, transformation opportunities, byproducts, and universal transformation graph language.

## Does Not Own

Navigation choice, action execution, storage infrastructure, or reality preservation.

## Inputs

Assets in context, constraints, capabilities, relationships, objectives, and evidence.

## Outputs

Transformation options, transformation paths, affordance maps, and opportunity surfaces.

## Dependencies

ARK, WEAVE, Asset Contracts, Capability Contracts, CivPhys.

## Consumers

Jarvis, Foundry, domains, and Reasoning.
