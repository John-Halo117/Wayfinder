# Contributing

All changes must pass the Polaris Constitutional Compliance Framework.

Before proposing a change, verify:

- Wayfinder's mission remains unchanged;
- semantic ownership remains limited to approved Wayfinder capabilities;
- no runtime dependency is introduced;
- no public contract, mission, ownership, or dependency rule changes without
  ADR approval;
- Commons remains the mechanical publisher of contract artifacts;
- ARK remains the durable evidence and persistence owner;
- Observatory remains the operational telemetry owner.
