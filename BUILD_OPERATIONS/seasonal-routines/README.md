# Seasonal Routines

Seasonal routines describe recurring property operations tied to weather,
growth, access, water, energy, and maintenance seasons.

## Purpose

Make seasonal transitions explicit and repeatable.

## Required Routine Fields

- routine ID
- season or trigger
- affected scope IDs
- pre-checks
- actions
- acceptance checks
- service records to create
- Build Bible records to review

## Relationships

- Maintenance schedules: [Maintenance Schedules](../../BUILD_BIBLE/lifecycle/maintenance-schedules/README.md)
- Reliability inspections: [Inspection Standards](../../BUILD_BIBLE/lifecycle/reliability/inspection-standards.md)
- Observations: [Observations](../../BUILD_BIBLE/reality/observations/README.md)

## Generation Targets

Seasonal checklists, calendar reminders, inspection routes, operator packets.
