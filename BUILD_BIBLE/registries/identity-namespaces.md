# Identity Namespaces

Stable IDs use `bb:<namespace>:<slug>`.

## Namespaces

| Namespace | Purpose |
| --- | --- |
| `world` | Top-level physical roots. |
| `land` | Parcels, terrain, soil, and natural site features. |
| `site` | Site planning, access, zones, exterior infrastructure. |
| `building` | Buildings and major building assemblies. |
| `space` | Rooms, zones, cabinets, closets, and occupiable or service spaces. |
| `utility` | Shared utility infrastructure. |
| `water` | Potable, non-potable, storage, treatment, and distribution. |
| `drainage` | Stormwater, grading, drains, swales, retention, and discharge. |
| `waste` | Wastewater, solid waste, compost, recycling, and hazardous waste handling. |
| `electrical` | Electrical service, generation, storage, distribution, and loads. |
| `low-voltage` | Low-voltage wiring, control, sensors, and signaling. |
| `network` | Physical networking infrastructure. |
| `hvac` | Heating, ventilation, air conditioning, filtration, and distribution. |
| `lighting` | Lighting fixtures, circuits, controls, and scenes. |
| `security` | Physical security, access control, surveillance, and alarms. |
| `storage` | Physical storage systems and inventories. |
| `workshop` | Shops, tool zones, fabrication areas, and service benches. |
| `agriculture` | Gardens, orchards, greenhouses, food forests, and soil systems. |
| `livestock` | Animal housing, fencing, water, feed, and handling infrastructure. |
| `road` | Roads, driveways, paths, bridges, gates, and vehicle circulation. |
| `equipment` | Tools, vehicles, machines, appliances, and maintainable assets. |
| `material` | Materials, finishes, assemblies, and approved products. |
| `spine` | Service spines at any scale. |
| `interface` | Connections between scopes. |
| `capability` | Physical capabilities exposed by scopes. |
| `maintenance` | Maintenance procedures and schedules. |
| `observation` | Bounded claims about observed physical reality. |
| `decision` | Durable physical design and lifecycle decisions. |
| `artifact` | Generated artifact manifests. |
| `resource-flow` | Resource-flow records. |
| `dependency` | Dependency records. |
| `reliability` | Reliability records and failure/degradation models. |
