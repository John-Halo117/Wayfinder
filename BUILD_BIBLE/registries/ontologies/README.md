# Ontologies

Ontologies define reusable classification and relationship models for physical
systems.

## Purpose

Compress common physical reasoning into stable vocabularies that every domain
can reference.

## Relationships

Ontologies support [Universal Scope Contract](../../contracts/universal-scope-contract.md),
domain patterns, reliability records, generated diagrams, and future digital
twin reasoning.

## Lifecycle

Ontology changes require an Engineering Decision Record when they alter the
meaning of existing records.

## Generation Targets

Dependency graphs, capability maps, resource-flow diagrams, digital twin
exports, review reports, and reliability maps.
