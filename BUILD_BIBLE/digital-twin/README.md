# Digital Twin

The digital twin is the structured representation of physical scopes and their
current known state.

## Owns

- digital-twin semantics
- spatial addressing model
- current state model
- service history model
- observation linkage
- last verified reality semantics
- digital twin readiness semantics

## Does Not Own

- Generated 3D models.
- Generated Home Assistant configs.
- Generated ESPHome configs.
- Runtime telemetry storage.

Generated exports belong in `generation/`.
