# Interface Contract

An interface is a defined connection between physical scopes.

## Required Fields

- interface ID
- provider scope ID
- consumer scope ID
- medium
- capacity
- connector or termination
- isolation method
- failure behavior
- access method
- verification state

## Examples

- electrical circuit to an appliance
- water branch to a fixture
- network cable to an access point
- ventilation duct to a room
- drainage connection to a stack
