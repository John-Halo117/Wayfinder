# Inspection Standards

Inspection standards define how condition is checked before acceptance,
operation, maintenance, renewal, or retirement.

## Required Fields

- inspected scope or capability class
- inspection trigger
- required access
- inspection method
- required evidence
- pass criteria
- fail criteria
- escalation path
- service record location

## Rule

Inspection must be possible through documented access paths. If a critical
condition cannot be inspected, the design must link to an Engineering Decision
Record accepting that constraint.
