# Maintenance Schedules

Maintenance schedules define recurring obligations.

Each schedule should include:

- maintained scope ID
- interval or trigger
- procedure link
- responsible role
- required access
- expected duration
- pass criteria
- escalation path

## Purpose

Convert reusable maintenance procedures into scheduled obligations.

## Contents

- [Reusable Frequency Classes](reusable-frequency-classes.md)
- scope-specific recurring schedules
- triggers

## Relationships

## Lifecycle

Schedules change when physical systems, observed risk, manufacturer
requirements, or lifecycle decisions change.

## Generation Targets

Calendars, reminders, inspection checklists, service work orders, and digital
twin synchronization tasks.
