# Lifecycle

Lifecycle records preserve why and how physical reality changes over time.

## Owns

- property lifecycle phases
- decisions
- changes
- reliability
- service records
- maintenance schedules
- retirement records

Lifecycle records link to affected physical scopes instead of replacing scope
documents.
