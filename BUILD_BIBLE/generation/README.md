# Generation

Generation contains derived outputs from the canonical Build Bible.

Generated artifacts are non-canonical and disposable. They must identify their
source scopes, source revisions, generator, generator version, timestamp, and
validation status.

Generated artifact manifests must conform to
[`../schemas/generated-artifact-manifest.schema.json`](../schemas/generated-artifact-manifest.schema.json).

## Output Classes

- CAD
- BIM
- diagrams
- bills of materials
- material takeoffs
- construction phases
- inspection checklists
- maintenance schedules
- Home Assistant configs
- ESPHome configs
- network configs
- documentation
