# Reserve Capacity

Reserve capacity preserves maneuverability.

## Capacity Types

- physical space
- load capacity
- flow capacity
- electrical capacity
- thermal capacity
- data capacity
- drainage capacity
- access capacity
- maintenance capacity

## Rule

Each scope should publish current use, maximum safe use, reserved capacity, and
known bottlenecks.

## Capacity Is Not Waste

Unused conduit, spare breaker positions, accessible chases, pull strings,
extra mounting surfaces, blank panels, and service clearances are deliberate
future capability.
