## Rule

Indexes point to canonical owners. They do not restate canonical facts.

Indexes are the first retrieval surface for Build Bible navigation. Tools
should consult indexes and metadata before scanning full canonical documents,
and should escalate only when the requested objective cannot be satisfied with
adequate confidence.

## Generation Targets

Navigation pages, knowledge graph entry points, query indexes, review packets,
and compiler manifests.
