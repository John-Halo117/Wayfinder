# Drainage Domain

Owns stormwater, roof drainage, grading interfaces, swales, drains, culverts,
retention, detention, infiltration, and discharge.

Canonical here:

- drainage paths
- inspection ports
- cleanouts
- capacity constraints
- overflow behavior
- erosion controls

## Purpose

Define how water moves, is slowed, stored, infiltrated, discharged, inspected,
and maintained.

## Lifecycle

Drainage patterns are reusable. Actual routes are verified through surveys,
storm observations, inspections, and service records.

inspection checklists.
