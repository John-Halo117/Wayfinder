# Site Domain

Owns site planning above the land layer: zones, circulation, access, exterior
utility corridors, service yards, staging areas, outdoor rooms, and expansion
reserves.

Canonical here:

- property spine
- site zones
- exterior access routes
- service corridors
- future expansion zones

## Purpose

Define the property-scale operating platform that coordinates land, access,
utilities, drainage, observation, and expansion.

## Lifecycle

Site patterns are reusable. Property-specific surveys, decisions, and
observations instantiate them.

## Generation Targets

Site plans, utility ingress maps, drainage overlays, access plans, expansion
reserve maps, and inspection routes.
