# Reports

Reports define reusable report templates for repository quality and coverage.

P3 defines report shapes. It does not implement report generation.

## Lifecycle

Report templates change when repository quality criteria or validation rules
change.

## Generation Targets

Architecture health reports, repository health reports, completeness reports,
traceability reports, readiness reports, and maintainability reports.
