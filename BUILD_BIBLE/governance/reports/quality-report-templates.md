# Quality Report Templates

Quality reports evaluate repository health and architectural maintainability.

## Quality Report Types

- architecture health
- pattern health
- repository health
- documentation completeness
- relationship density
- composition complexity
- traceability health
- validation health
- maintainability health

## Required Report Fields

- report ID
- report type
- evaluated layer or directory
- quality criteria
- findings
- severity
- affected documents
- recommended action
- accepted tradeoffs
- related EDRs

## Rule

Quality reports are diagnostic artifacts. They do not become canonical design
truth unless a finding is promoted into a source document or decision record.
