# Repository Intelligence

Repository intelligence makes the Build Bible self-describing, traceable, and
composable.

## Purpose

Treat every document as a first-class engineering artifact that can be
discovered, queried, reviewed, validated, and used by future generators.

Repository intelligence follows Wayfinder Progressive Discovery: index and
metadata first, relationships before exhaustive traversal, targeted retrieval
before full document access, and review queues for low-confidence results.

artifact expectations.
