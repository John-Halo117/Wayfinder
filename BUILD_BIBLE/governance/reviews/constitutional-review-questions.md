# Constitutional Review Questions

Every major physical document should answer these questions directly or link to
the record that answers them.

## Questions

- Why does this exist?
- What capability does it provide?
- What depends upon it?
- What does it depend upon?
- How does it fail?
- How is failure detected?
- How is it serviced?
- How is it upgraded?
- How is it replaced?
- What eventually succeeds or replaces it?

## Rule

The questions are a documentation discipline. They preserve living architecture
by keeping each addition connected to the broader physical system.
