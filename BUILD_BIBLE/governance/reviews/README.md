# Reviews

Reviews define reusable engineering review tools for physical systems.

## Purpose

Evaluate designs against canonical doctrine before construction, purchase,
commissioning, operation, expansion, renewal, or retirement.

## Relationships

Reviews consume doctrine, contracts, schemas, ontologies, reliability records,
and lifecycle decisions.

## Lifecycle

Review tools change through Engineering Decision Records when evaluation
meaning changes.

## Generation Targets

Design reviews, acceptance checklists, commissioning packets, traceability
reports, and maintenance review dashboards.
